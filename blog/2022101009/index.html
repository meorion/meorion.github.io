CVE-2022-3176 addresses a a use-after-free (happens in dynamic aka heap memory) bug/exploit in the io_uring of the Linux Kernel.

use-after-free, If a data set is deleted or moved to another block but the pointer, instead of being cleared (set to null), continues to refer to the now-freed memory, the result is a dangling pointer.
